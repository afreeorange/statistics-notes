# My Statistics Notes

For my classes at Georgia Tech. Powered by `vim`, [11ty](https://www.11ty.dev/), S3, and CircleCI.

## Working on Notes

```bash
# Install dependencies
yarn

# Start live-reloading server
yarn start

# Build (output in _site)
yarn build

# Commit with a lazy message and push to remote
yarn push
```
