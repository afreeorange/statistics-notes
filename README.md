# My Statistics Notes

For my classes at Georgia Tech. Powered by `vim`, [11ty](https://www.11ty.dev/), S3, and Github Actions.

## Working on Notes

```bash
# Install dependencies
yarn

# Start live-reloading server
yarn start

# Build (output in _site)
yarn build

# Deploy to S3_BUCKET in .env
yarn deploy
```
